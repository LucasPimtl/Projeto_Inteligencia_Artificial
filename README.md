# Projeto_Inteligencia_Artificial

Projeto desenvolvido na matéria de Inteligência Artificial. O projeto buscou comparar o funcionamento do algoritmo de aprendizado profundo LSTM com o ARIMA (AutoRegressive Integrated Moving Average) e também com a Regressão Linear no contexto de previsão de preços de ativos usando YahooFinance. Como foi constatado que buscando prever somente o fechamento do dia seguinte, o LSTM desempenhava pior, foi usado uma alternativa para previsão de variações nos 2 últimos notebooks.
